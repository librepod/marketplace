3. Extract `spec.secrets` block: Capture from `  secrets:` to the next sibling key.

Since the current script uses grep/sed without yq, and the template blocks are multi-line YAML literals, the most reliable approach is to use `awk` to extract sections between known markers.

Add these extraction functions before the main loop:

```bash
# Extract a YAML section from spec level (2-space indent) to the next spec-level key
extract_section() {
  local file="$1"
  local section="$2"
  awk -v section="  $section:" '
    $0 ~ "^" section { found=1; print; next }
    found && /^  [a-zA-Z]/ { exit }
    found { print }
  ' "$file"
}
```

In the catalog entry template (the `cat >> "$CATALOG_FILE" <<ENTRY` block), after the existing fields, append:

extract_yaml_block() {
